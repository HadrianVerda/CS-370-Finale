The majority of the AI’s methods were pre-created for this assignment. My contribution handled primarily and only the code regarding the functionality of the AI algorithm. How do I approach a problem as a computer scientist?
For any problem I encounter with software my first instinct is to isolate and determine the general area the issue is affecting then research possible solutions to this issue. If a solution has been found then I see if it works for this particular issue if a solution has not been discovered, yet I then start developing a solution.

As a computer scientist I have a responsibility to develop safe code and ensure the privacy of users is maintained and protected across my software.
